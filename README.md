# Deep Learning for Computer Vision and NLP

## Description

Ce projet présente l'implémentation et l'évaluation comparative de plusieurs architectures de Deep Learning : MLP, CNN, RNN, LSTM, GRU et Seq2Seq.

Les expériences réalisées portent sur :

- Classification d'images avec MNIST
- Classification d'images avec Fashion-MNIST
- Analyse de sentiments
- Traduction automatique neuronale

## Architectures implémentées

- MLP
- CNN
- RNN
- LSTM
- GRU
- Seq2Seq

## Techniques avancées

- Teacher Forcing
- Gradient Clipping
- Beam Search
- BLEU Score

## Résultats

| Modèle | Performance |
|---------|------------|
| RNN | Accuracy = 0.7704 |
| LSTM | Accuracy = 0.8126 |
| GRU | Accuracy = 0.8794 |
| Seq2Seq | BLEU = 0.4085 |
